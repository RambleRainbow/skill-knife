# Contributing to SkillKnife

Thank you for your interest in contributing to SkillKnife!

## Getting Started

1.  **Fork** the repository on GitHub.
2.  **Clone** your fork locally:
    ```bash
    git clone https://github.com/YOUR_USERNAME/skill-knife.git
    cd skill-knife
    ```
3.  **Install Dependencies**:
    ```bash
    npm install
    ```

## Development Workflow

1.  Open the project in VS Code.
2.  Press `F5` to launch the **Extension Development Host**.
3.  Changes to source files will require a reload of the host window (`Cmd+R` in the host).

## Code Style

- We use **ESLint** and **Prettier** for code formatting.
- Run `npm run lint` before committing to ensure code quality.

## Submitting Changes

1.  Create a new branch for your feature/bugfix:
    ```bash
    git checkout -b feature/my-new-feature
    ```
2.  Commit your changes with clear messages.
3.  Push to your fork and submit a **Pull Request** to the `main` branch.
